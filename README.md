# PDF Reviewer

This project is a Python script for reviewing PDF files. It allows you to extract text from PDFs and add comments. 

## Features
- Extract text from PDF files
- Add comments to PDF pages
- Modular and well-documented code

## Requirements
- Python 3.8+
- PyPDF2 or pdfplumber

## Usage
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the main script:
   ```bash
   python main.py
   ```
